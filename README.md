# C# WinForm 음악 플레이어

C# Windows Forms를 이용해 만든 음악 재생 프로그램입니다.

음악 재생과 일시정지, 이전 곡과 다음 곡 이동, 볼륨 조절, 재생목록 확인 등의 기본 기능을 구현했습니다.  
또한 재생 중인 음악의 소리를 분석해 주파수에 따라 그래프가 움직이는 오디오 시각화 기능을 추가했습니다.

- **프로젝트 기간:** 2026.03.05 ~ 2026.03.17
- **개발 환경:** C#, Windows Forms, Visual Studio 2022
- **사용 도구:** NAudio, FFTSharp

---

## 프로젝트 소개

사용자가 음악을 선택하면 프로그램 안에서 음악을 재생하고, 재생 중인 음악의 소리 크기와 주파수 변화가 그래프로 표시되도록 만든 프로젝트입니다.

팀 프로젝트 전체에는 로그인, 회원가입, 음악 차트, 뉴스, 관리자 기능 등이 포함되어 있으며, 저는 그중에서 **메인 음악 플레이어 화면과 오디오 그래프 기능**을 담당했습니다.

---

## 메인 화면

음악 재생 화면, 재생목록, 최근 재생 목록, 많이 들은 음악 등의 기능을 한 화면에서 확인할 수 있도록 구성했습니다.

![메인 화면](./메인%20화면.png)

---

## 음악 재생 화면

앨범 이미지와 노래 제목을 표시하고, 음악 재생에 필요한 기본 버튼을 배치했습니다.

구현한 기능은 다음과 같습니다.

- 음악 재생 및 일시정지
- 이전 곡 및 다음 곡 이동
- 반복 재생
- 재생 위치 확인 및 변경
- 볼륨 조절
- 음악에 맞춰 움직이는 그래프

![재생 화면](./재생%20화면.png)

---

## 음악 재생목록

음악 제목, 가수, 앨범 이미지와 재생 시간을 목록으로 표시했습니다.

현재 선택한 음악이 다른 색으로 표시되도록 만들어 사용자가 어떤 음악을 선택했는지 쉽게 확인할 수 있도록 했습니다.

![음악 재생 목록](./음악%20재생%20목록.png)

## 실시간 음악 그래프

재생 중인 음악의 소리를 받아 주파수 값을 구한 뒤, 값의 크기에 따라 막대그래프의 높이가 변하도록 구현했습니다.

음악의 소리가 클수록 그래프가 크게 움직이고, 소리가 작을수록 그래프가 작게 움직이도록 만들었습니다.

![음악 주파수](./음악%20주파수.png)

---

## 음악 그래프 구현

NAudio를 이용해 재생 중인 음악 데이터를 가져오고, FFTSharp를 이용해 음악 데이터를 주파수 값으로 바꿨습니다.

바뀐 주파수 값을 그래프 높이에 적용해 음악이 재생되는 동안 그래프가 계속 움직이도록 구현했습니다.

```text
음악 재생
    ↓
NAudio로 음악 데이터 받기
    ↓
FFT를 이용해 주파수 값으로 변환
    ↓
주파수 값에 따라 그래프 높이 변경
